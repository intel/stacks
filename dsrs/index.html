Data Services Reference Stack

Sysstacks containers have been deprecated and please switch to oneapi based containers, you can find oneapi containers at this link : https://hub.docker.com/u/intel The Data Services Reference Stack (DSRS) combines the best-in-class database and data analytics management apps. Featuring Intel® Optane™ DC persistent memory, DSRS helps you to store and process large amounts of data at unprecedented speeds using a distributed processing framework.

To offer more flexibility, there are multiple versions of the Data Services Reference Stack. Each version includes instructions for using, building and configuring the stack to help get the most out of it.
